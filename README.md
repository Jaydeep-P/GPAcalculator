# GPA Calculator

A simple web application used to calculate GPA based on credits and grades.

# Tech Stack

NextJS
Node.js
MongoDB

## How to run

    git clone https://github.com/hrushikesh-02/GPAcalculator.git
    npm i
    npm run dev
